## Overview

This project is a full-stack personal profile application built using **Node.js, Express, MongoDB, and React**.

The backend exposes REST APIs to:

- Fetch profile information
- List projects
- Search projects by skill
- Retrieve top skills
- Perform keyword-based search

The frontend consumes these APIs and provides a simple UI to view the profile, projects, and search functionality.
